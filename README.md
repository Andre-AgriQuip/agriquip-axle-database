# AgriQuip Axle Bracket Finder - Static Version

This is the simple static version for Vercel.

## Upload to GitHub

Upload these items directly to the repository root:

- index.html
- data/records.json
- README.md

Do not upload the outer folder itself.

## Deploy to Vercel

Import the GitHub repository into Vercel and deploy. Because this version has an index.html in the root, Vercel can serve it as a static website.

## Notes

This version is searchable and view-only. File uploads for photos, PDFs and DXFs need Vercel Blob or another storage option added later.
